2020 新的开始

发表于2020-01-07|更新于2020-01-14
|阅读量:

前言

新的一年，新的起点，新的开始。

博客更换

之前是使用Wordpress博客，不过由于不怎么写博客，Wordpress就太重了一点，还有一个问题就是Wordpress的PHP有时候会无响应，只能重启php。

hexo就是一个很好的选择了，对我这种几个小时打鱼几个月晒网的人，hexo就很轻量，静态博客不用怎么管理，想写就写，Markdown格式通用且迁移方便。

使用的是一个简单优雅的主题Suka Theme

发现一个超好看的博客蝉時雨，好看的博客系统需要好好维护，我这种太懒的人不行。

使用GitBook

最开始使用为知笔记，还充过会员，不过感觉软件一直没有什么进步，还一个明显的退出奔溃Bug一年没修过，感觉跟没测试的人一样。

后来听说Notion很好用，就转战Notion，把为知的笔记都导进去了，block写法很怪，不习惯。
缺点客户端是全网页的，速度很慢，一个简单的table都要插入一个数据库。
导出的Markdown的格式很乱，文件名几乎全是Untitled，没法好好迁移备份，也不是正规的Markdown格式。

现在转到GitBook了，纯Markdown笔记，格式通用，书写方便，还能和Github同步。

组装一台新的电脑

准备年后回来配一个高配的台式机，等10代Intel CPU， AMD CPU每一代性能提升太多，旧的不保值。Intel CPU每一代基本没啥提升，所以保值一点，性能价格都比不上AMD。
AMD什么都好，但AMD CPU不好装黑苹果系统，因为苹果电脑一直用Intel CPU，买和白苹果配置相同更容易黑苹果。
